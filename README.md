A creative brainstorming tool for exploring vibe coding project ideas.

## What is "Vibe Coding"?

Vibe coding is about creating projects that prioritize aesthetic, feeling, and experience over pure functionality. It's code that makes you *feel* something - whether that's:
- Calming and meditative
- Energizing and dynamic
- Nostalgic and retro
- Futuristic and sleek
- Playful and whimsical

## How to Use This Tool

1. Browse `ideas/` directory for inspiration
2. Use `brainstorm.md` for freeform ideation
3. Capture structured ideas in `ideas/` using the template
4. Iterate and combine concepts

## Project Categories

- **Visual Vibes**: Generative art, animations, visual experiences
- **Audio Vibes**: Soundscapes, interactive music, audio-reactive visuals
- **Interactive Vibes**: Games, toys, experimental UIs
- **Ambient Vibes**: Background experiences, screensavers, environmental tools
- **Data Vibes**: Visualizing data in aesthetic ways
- **Utility Vibes**: Practical tools with exceptional aesthetics

## Get Started

Run through the prompts in `brainstorm.md` or dive straight into creating idea files!
